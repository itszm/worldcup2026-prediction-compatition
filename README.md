# 2026 FIFA World Cup Predictions & Bracket Comparison Hub

这是一个 2026 美加墨世界杯赛程和预测的对照与导航系统。本项目非常独特——**所有的 Next.js 子项目及导航页面全部由不同的 AI 辅助编程工具独立开发完成**。

本项目包含三个独立的 Next.js 世界杯预测应用，并提供了一个统一的主入口导航门户，用于展示和比较各 AI 模型的预测结果。

---

## 🚀 项目结构

本仓库的根目录下包含以下三个子项目：

1. **`antigravity/`** (由 **Antigravity - Gemini 3.5 Flash (High)** 开发)
   * **预测冠军**：🇦🇷 阿根廷 (15.0%)
   * **特色**：Elo 动态实力评分演算法、12个小组的模拟积分表、淘汰赛对阵图、比赛多维度交互打分及 MVP 评选。
2. **`claude/`** (由 **Claude Code - Fable 5 (High)** 开发)
   * **预测冠军**：🇪🇸 西班牙 (16.5%，决赛击败阿根廷的概率为 54%)
   * **特色**：官方分组及赛程安排、可视化四强/决赛对阵路径、金靴奖得主概率雷达（姆巴佩、哈兰德等）。
3. **`codex/`** (由 **Codex - GPT 5.5 (High)** 开发)
   * **预测冠军**：🇪🇸 西班牙 (19.0%，决赛 2-1 击败法国)
   * **特色**：交互式 Match Lab 评分实验室（自选 5 因子加权模型）、20,000 次 Monte Carlo 模拟的小组晋级看板。

---

## 📂 导航主页 (Portal)

为了方便访问和比较这三个项目，我们在主目录下提供了三个**独立版本的导航入口网页**。您可以根据部署环境任选其一：

* **`index.html` (本地独立端口模式 - Dev)**
  * 默认链接至 `localhost:3001` (Antigravity)、`3002` (Claude) 和 `3003` (Codex)。
* **`index-subpath.html` (Nginx 子路径反代模式)**
  * 适用于单域名下通过子路径反向代理访问，默认链接至 `/antigravity/`、`/claude/` 和 `/codex/`。
* **`index-custom.html` (自定义域名/链接模式)**
  * 适用于您将三个服务解析到不同的自定义域名。

> 💡 **如何自定义链接**：这三个 HTML 页面的顶部都包含一个易于修改的 `<script>` 配置块，您可以直接用文本编辑器打开并手动修改其中的服务链接。

## ⚙️ 本地运行指南

我们在根目录下提供了一个快捷启动脚本 `start-all.sh`，可一键同时启动三个 Next.js 服务以及导航主页：

1. **赋予脚本执行权限**：
   ```bash
   chmod +x start-all.sh
   ```
2. **运行脚本**：
   ```bash
   ./start-all.sh
   ```
   * 脚本会自动检查并安装子项目所需的 `node_modules` 依赖。
   * 启动三个服务分处 `3001`、`3002` 和 `3003` 端口。
   * 启动一个轻量级 Web 服务器在 `http://localhost:3000` 承载导航主页。
3. **退出运行**：
   在终端中按下 `Ctrl + C`，脚本会自动捕获信号并**优雅地关闭所有后台服务进程**。
